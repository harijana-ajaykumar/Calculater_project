# 🧮 Calculator Project

A simple Calculator Application developed using Python. This project performs basic arithmetic operations such as Addition, Subtraction, Multiplication, and Division.

## Features
- Addition
- Subtraction
- Multiplication
- Division
- Easy to use

## Technologies Used
- Python

## Requirements
- Python 3.x
- Any Python IDE (VS Code, PyCharm, IDLE, etc.)

## How to Run
1. Download or clone the repository.
2. Open the Python file.
3. Run the program.
4. Enter the required numbers and select an operation.
